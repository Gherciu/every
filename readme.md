## Installation

```bash
npm i -g every-cli
```

## Usage

```
$ every --help

  Usage
    $ every <command> [arguments]

  Examples
    $ every pwd
      - folder-1
        c:/my-projects/folder-1
      - folder-2
        c:/my-projects/folder-2
```
